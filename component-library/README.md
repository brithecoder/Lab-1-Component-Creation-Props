# Lab 1 Component Creation & Props

This template provides a minimal setup to get React working in Vite with HMR and some ESLint rules.I utilized Tailswind for the css. To run the app you want to cd into the component-library file and run npm run dev this will start the program on a local host port. 

## Reflection Questions

### How did you handle optional props in your components?

I handled optional props by using ternary statements to display the status or state of the 
elements. 

### What considerations did you make when designing the component interfaces?

I used what would be displayed for in the end product for designing the interfaces and also the template given to us for the lab. 

### How did you ensure type safety across your components?

I insured type saftey by giving most of the props there own type created by the interfaces.

### What challenges did you face when implementing component composition?

I faced challenges with figuring out how to utilize the template given to us and also putting the components together.
